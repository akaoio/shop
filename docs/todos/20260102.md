Còn 1 điều quan trọng: làm sao có danh sách tất cả các items, và items nào cần để render?

==> Quyết định kiến trúc:
- src/statics/system.yaml -> build/statics/system.json: có thiết lập cơ bản cấp hệ thống như số con trên mỗi page được chia pagination: 10 (10 con trên mỗi page)
- Lấy danh sách items -> tạo /build/statics/items/meta.json và 1.json 2.json v.v... -> chứa đường dẫn để query items
- Lấy danh sách tags -> tạo /build/statics/tags/meta.json và 1.json 2.json v.v... -> chứa danh sách tags, chia theo từng file
- Với từng tag -> tạo /build/statics/tags/[tag]/meta.json chứa danh sách item mà tag đó được gán, 1.json 2.json... chứa danh sách tương tự nhưng chia từng file theo số pagination
